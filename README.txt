How to start application

1. Open cmd
2. go to root folder
3. go to /venv/scripts
4. "activate.bat"
5. go back to root
6. "py ChessEngine.py"

Preko komandne linije pozicionirati se u folder /venv/scripts
ukucati komandu "activate.bat"
vratiti se nazad u koreni direktorijum
ukucati "py ChessEngine.py"
